إعداد Firebase تلقائياً

أدخل بيانات Firebase — وسنطبقها تلقائياً على التطبيق ونبني APK جديد

📊 تقدم الإعداد

0% مكتمل
1 بيانات Web App (firebaseConfig)
2 ملف google-services.json
3 VAPID Key للـ Web Push
4 Service Account JSON (لإرسال الإشعارات)
5 تطبيق البيانات وبناء APK

📋 خطوات إنشاء مشروع Firebase

💡 اتبع الخطوات بالترتيب. كل خطوة تأخذ 1-2 دقيقة فقط.

📌 الخطوة 1: إنشاء مشروع Firebase

1 اذهب إلى console.firebase.google.com
2 اضغط "Add Project"
3 اكتب اسم المشروع: nawaqes-app
4 اضغط Continue → عطّل Google Analytics → Create Project
5 انتظر 30 ثانية، ثم اضغط Continue

📌 الخطوة 2: إضافة تطبيق Web

1 من الصفحة الرئيسية، اضغط أيقونة </>
2 App nickname: Nawaqes Web
3 اضغط Register app
4 سترى كود JavaScript مع firebaseConfig — انسخه كاملاً
5 انتقل لتبويب "🔐 بيانات Firebase" والصق الكود

📌 الخطوة 3: إضافة تطبيق Android

1 من Project Settings: Add app → Android
2 Android package name: com.nawaqes.app
3 App nickname: Nawaqes
4 اضغط Register app
5 حمّل ملف google-services.json
6 انتقل لتبويب "📦 google-services.json" وارفع الملف

📌 الخطوة 4: إنشاء VAPID Key

1 من القائمة: Engage → Cloud Messaging
2 اذهب لتبويب Web Configuration
3 تحت Web Push certificates اضغط Generate key pair
4 انسخ الـ Key pair (يبدأ بـ BJ...)
5 انتقل لتبويب "🔔 VAPID Key" والصقه

📌 الخطوة 5: Service Account

1 اذهب إلى Project Settings → Service Accounts
2 اضغط Generate new private key → Generate key
3 سيُحمّل ملف JSON
4 انتقل لتبويب "⚙️ Service Account" وارفع الملف

🔐 بيانات Firebase Web Config

الصق كود JavaScript بالكامل الذي حصلت عليه من Firebase Console (يحتوي على apiKey, authDomain, ...).

📋 الصق الكود أعلاه ليتم استخراج القيم تلقائياً

✅ تم استخراج القيم بنجاح:

📦 رفع google-services.json

ارفع ملف google-services.json الذي حمّلته من Firebase Console.

📦 اختر الملف للتحقق منه تلقائياً

✅ ملف صحيح:

🔔 VAPID Key (Web Push)

الصق الـ VAPID Key الذي أنشأته من Cloud Messaging → Web Configuration.

🔔 الصق الـ VAPID Key

⚙️ Service Account JSON

ارفع ملف Service Account JSON (للسماح للسيرفر بإرسال الإشعارات).

⚠️ تنبيه أمني مهم: هذا الملف يحتوي على صلاحيات إدارة المشروع! لا تشاركه علناً.
⚙️ اختر الملف للتحقق منه

✅ ملف صحيح:

🚀 مراجعة وتطبيق البيانات

راجع جميع البيانات قبل التطبيق، ثم حمّل ملف الإعداد الكامل.
